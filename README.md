# Rat_scarred_uterus_analysis_project
The code for this project is used to analyze tissue-level mRNA-seq data of full-thickness uterine tissue at the incision site in rats, collected at three different postoperative time points (Day 3, Day 7, and Day 20) following full-thickness injury.
